# Flask Mini App

A simple web application built with Flask to showcase that I can build and run a basic Python web app.

# Features

- Home page (`/`) with a short description of my portfolio projects  
- About page (`/about`) rendered from a separate HTML template  
- Uses Flask and Jinja2 templates

# Skills Practiced

- Flask project structure (`app.py`, `templates/`)  
- Defining routes with `@app.route`  
- Rendering HTML templates with `render_template`

# How to Run


python3 app.py
