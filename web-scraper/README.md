# A simple multi-page web scraper built with     Requests, BeautifulSoup, and Pandas.

 # What the scraper does
	•	Scrapes book titles and prices
	•	Loops through multiple pages
	•	Cleans the data
	•	Converts prices to numeric values
	•	Saves results to CSV
	•	Prints a short analysis (top 5 expensive + cheap books)

 # Technologies
	•	Python
	•	Requests
	•	BeautifulSoup4
	•	Pandas

 # Output
	•	books_all_pages.csv — cleaned dataset
	•	printed analysis in terminal
